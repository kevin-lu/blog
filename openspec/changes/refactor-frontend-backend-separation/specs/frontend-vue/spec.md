## ADDED Requirements

### Requirement: 博客展示前端

系统应提供 Vue 3 构建的博客展示前端，支持响应式布局，适配桌面和移动设备。

#### Scenario: 访问首页
- **WHEN** 用户访问网站首页
- **THEN** 系统展示最新文章列表、站点介绍、导航菜单

#### Scenario: 访问文章详情页
- **WHEN** 用户点击文章卡片或链接
- **THEN** 系统展示文章完整内容、目录导航、评论区域、相关文章推荐

#### Scenario: 访问分类页
- **WHEN** 用户点击分类标签
- **THEN** 系统展示该分类下的所有文章列表

#### Scenario: 访问标签页
- **WHEN** 用户点击标签
- **THEN** 系统展示包含该标签的所有文章列表

#### Scenario: 主题切换
- **WHEN** 用户点击主题切换按钮
- **THEN** 系统在亮色/暗色主题之间切换，并保存用户偏好

### Requirement: 管理后台前端

系统应提供 Vue 3 构建的管理后台，支持文章管理、分类管理、标签管理、评论审核、站点设置。

#### Scenario: 管理员登录
- **WHEN** 管理员访问 `/admin/login` 并输入正确的用户名密码
- **THEN** 系统跳转到管理后台首页，并保存登录状态

#### Scenario: 创建文章
- **WHEN** 管理员点击"新建文章"，填写标题、内容、分类、标签后保存
- **THEN** 系统创建文章并提示保存成功

#### Scenario: 编辑文章
- **WHEN** 管理员在文章列表中选择一篇文章进行编辑
- **THEN** 系统加载文章详情，管理员修改后保存成功

#### Scenario: 删除文章
- **WHEN** 管理员在文章列表中点击删除按钮并确认
- **THEN** 系统删除文章并从列表中移除

#### Scenario: 管理分类
- **WHEN** 管理员在分类管理页面添加新分类
- **THEN** 系统创建分类并更新分类列表

#### Scenario: 审核评论
- **WHEN** 管理员在评论管理页面审核评论
- **THEN** 系统更新评论状态，通过审核的评论在前台显示

### Requirement: 富文本编辑器

系统应集成富文本编辑器，支持文章编辑、图片上传、代码高亮。

#### Scenario: 编辑文章内容
- **WHEN** 管理员在文章编辑页面输入内容
- **THEN** 系统提供富文本编辑功能，支持格式化、插入图片、代码块等

#### Scenario: 上传图片
- **WHEN** 管理员在编辑器中上传图片
- **THEN** 系统上传图片并在编辑器中显示图片预览

#### Scenario: 插入代码块
- **WHEN** 管理员插入代码块并选择编程语言
- **THEN** 系统提供语法高亮显示

### Requirement: 响应式布局

系统应支持响应式布局，适配不同设备屏幕。

#### Scenario: 桌面端访问
- **WHEN** 用户使用桌面浏览器访问 (屏幕宽度 >= 1024px)
- **THEN** 系统展示三栏布局 (导航、内容、侧边栏)

#### Scenario: 平板端访问
- **WHEN** 用户使用平板设备访问 (768px <= 屏幕宽度 < 1024px)
- **THEN** 系统展示两栏布局，侧边栏折叠

#### Scenario: 移动端访问
- **WHEN** 用户使用手机访问 (屏幕宽度 < 768px)
- **THEN** 系统展示单栏布局，导航菜单折叠为汉堡菜单
